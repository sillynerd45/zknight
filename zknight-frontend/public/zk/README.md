# ZK Proving Assets

This directory contains the ZK proving artifacts needed for browser-based proof generation.

## Files

- **zknight.wasm** — Compiled circuit WebAssembly module (~800KB)
- **zknight_final.zkey** — Groth16 proving key (WARNING: large file, 100-500MB)

## Generation

These files are generated by the trusted setup process in `/zk/setup_trusted.sh`.

To regenerate:
```bash
cd zk
npm run setup
```

## Usage

The Web Worker (`/src/workers/prove.worker.ts`) loads these files to generate Groth16 proofs in the browser.

Expected proving time: 2-15 seconds depending on browser and device.

## Git Tracking

⚠️ **DO NOT commit `zknight_final.zkey`** — it's excluded via `.gitignore` due to size.

For production deployment, host the zkey file on a CDN and fetch it on demand.

The WASM file is small enough to commit or can also be CDN-hosted.
